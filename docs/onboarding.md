# Onboarding Questionnaire (MVP)

## Learner profile

- What is your current education level?
- What is your primary goal for this course?
- How many hours per week can you spend?
- Which topics are you already comfortable with?

## Personalization

- Which interests should examples use? (sports, business, music, etc.)
- Do you prefer concise explanations or deeper walkthroughs?
- How challenging should questions feel? (easy, balanced, stretch)

## Tutor character

- What tutor style do you want? (supportive coach, strict professor, calm mentor, energetic guide)
- Do you want direct correction, Socratic guidance, or mixed style?

## Initial state outputs

On completion, onboarding should update:

- `PROFILE.md`
- `PREFERENCES.md`
- `TUTOR_PERSONA.md`
